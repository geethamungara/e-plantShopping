# e-plantShopping

## Project Overview
e-plantShopping is a React-based online plant shopping application that allows users to browse and add plants to their shopping cart. The application is built using React and Redux for state management.

## Features
- Browse a wide variety of plants
- Add plants to shopping cart
- Remove plants from shopping cart
- View total cost of selected plants
- Responsive and user-friendly interface

## Technologies Used
- React JS
- Redux Toolkit
- CSS
- GitHub for version control

## Getting Started

### Installation
1. Clone the repository:
   git clone https://github.com/geethamungara/e-plantShopping.git

2. Navigate to project folder:
   cd e-plantShopping

3. Install dependencies:
   npm install

4. Run the application:
   npm start
